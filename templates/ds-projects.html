Titanic Project

In this project, I was keen on developing a simple supervised machine learning model to predict which passengers survived the titanic disaster. In particular, I wanted to build the following skills:

  • Data analysis including conducting statistical tests.
  • Training, evaluating and improving binary classifiers.
  • Modularising the model into a pipeline using PostgreSQL and Docker.
