Introduction

The main objective for this project was to create an ML classifier which could predict which passengers survived titanic disaster. Below are the steps I took to do this:

Project Plan

  1. Data engineering: downloading the data from Kaggle and importing into it a local PostgreSQL database.
  2. Data pre-processing: loading the data into a Jupyter environment and inspecting it for missing data.
  3. Data analysis: using statistical inference to find out what columns have a relationship with survival.
